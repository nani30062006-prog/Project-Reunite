# 🚀 Project Reunite
### AI-Powered Missing Person Identification System

Project Reunite is a cloud-based web application designed to simplify missing person case management. The system enables authorities to register, store, search, and manage missing person records through a secure and scalable AWS cloud infrastructure.

---

# 📌 Project Overview

The objective of Project Reunite is to digitize missing person records and provide a centralized platform for managing case information efficiently.

The application allows users to:

- Register missing person details
- Upload photographs
- Store records securely
- View and manage saved cases
- Search and retrieve case information
- Access the application through AWS Cloud

# ☁ AWS Services Used

### Amazon S3
Stores uploaded missing person images securely.

### Amazon DynamoDB
Stores all missing person records.

### Amazon EC2
Hosts the Flask backend application.

### Amazon CloudFront
Provides secure HTTPS access and faster content delivery.

### AWS IAM
Manages secure access and permissions.
